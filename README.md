# HexConstructor
This is the settlement planner for the BitCraft game. If you encounter any issues or bugs, please let me know.
- Controls:
  - Move around in WASD, rotate the camera with MMB + mouse.
  - In Landscape mode, right-click to raise a tile, left-click to lower a tile.
  - In Construction mode, right-click to place a building, left-click to remove a building, and Q and E to rotate the building.
- There's currently only one save, which works with the "Save" button.
- To close the application, press Alt+F4. XD
